# OpenMinds
Repo for the OpenMinds Toolkit prototyping board


This breakout board is based on the Raspberry-Pi Pico W board.

Connections include:
Haptic Motor : GP11
Neopixel LED : GP00
Button: GP22
LDR: GP26
Speaker: GP21
i2c 1 connector: SDA - GP14  -  SCL - GP15
i2c 0 connector: SDA - GP16  -  SCL - GP17
Servo Connector - GP18
